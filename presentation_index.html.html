技術說明

▸ 使用 HTML5 FileReader API 讀取本地 CSV，無需後端伺服器

▸ 欄位自動辨識：根據表頭關鍵字匹配（月份 / 營收 / 良率 / 客訴 / 產值）

▸ 缺失值處理：空值自動標記為 null，不影響其他欄位計算

▸ 支援拖曳上傳 + 點擊選檔 + 內建範例資料三種載入方式

▸ 右上角可隨時上傳新 CSV 替換資料，圖表自動重繪

AI 分析邏輯

▸ 五大面向：營收概況 → 良率分析 → 客訴狀況 → 人均產值 → AI 建議

▸ 趨勢偵測：自動比較本月 vs 上月 Delta，判斷上升 / 下降 / 持平

▸ 打字機效果：逐字呈現，提升演示效果
